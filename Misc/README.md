# Miscellaneous

Stuff that doesn't fit neatly into either module but is worth knowing about. Some of it is directly relevant to the course, some of it is just good to have in your feed if you're interested in tech.

---

## YouTube Channels

| Channel | What to expect |
|---------|----------------|
| [Computerphile](https://www.youtube.com/@Computerphile) | University of Nottingham professors explaining everything from cryptography to how the internet works - directly relevant to both modules |
| [Fireship](https://www.youtube.com/@Fireship) | Quick, dense breakdowns of tech trends and tools - the Code Report series is a good way to stay aware of what's happening in the industry |
| [Tom Scott](https://www.youtube.com/@TomScottGo) | Curious, well-researched videos on technology, language, and how things work - less technical but genuinely interesting |
| [Linus Tech Tips](https://www.youtube.com/@LinusTechTips) | Hardware, builds, and consumer tech - useful if you want to understand the physical side of what you're studying |

## Courses & Tools

**[Create with Code – Unity Learn](https://learn.unity.com/course/create-with-code)**

Free official Unity course. You build games in C# - which is close enough to what you're learning in CST1510 that the logic carries over. Good for making programming feel less abstract.

## Newsletters

**[Tom Scott's Newsletter](https://www.tomscott.com/newsletter/)**

Short, infrequent, and always worth reading. Extra context behind his videos and things he finds interesting. Low noise.
